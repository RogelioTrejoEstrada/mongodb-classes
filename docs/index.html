¿Que es NoSQL?

Conocidas como bases de datos no relacionales (Not Only SQL o No SQL), son un tipo de DB en la cual se modela datos de una manera especifica y con un modelo felxible, para la creación de aplicaciones modernas. Este tipo de base de datos es reconocida por su facil implementación y rendimeinto de escalabilidad.

Diferencias entre SQL y NoSQL

Característica SQL NoSQL
Forma de guardar los datos Tablas, Campos, Registros Colecciones, Documentos, Campos
Modelo de los datos Estricta al guardar los datos Flexible al guardar los datos
Lenguaje DDL, DML, DCL, TCL Cuenta con su propio lenguaje

Diferencias entre SQL y NoSQL

Característica SQL NoSQL
Rendimiento Depende del creador y de la cantidad de datos (en algun punto) Dependera del tipo y motor que se ocupe
Escalabilidad Sera mas vertical que horizontal Sera mas horizontal que vertical

Ventajas y desventajas de NoSQL

Ventajas

  • No genera cuellos de botella
  • Variedad para diferentes necesidades
  • Modelaje flexible
  • Volumenes gigantes
  • Fácil de implementar
  • Fácil de escalar
  • Rápido en consultas
  • Acepta redundancia

Ventajas y desventajas de NoSQL

Desventajas

  • Flexibilidad!!!!
  • Manejo para grandes volumenes de datos
  • No siempre es una buena opción
